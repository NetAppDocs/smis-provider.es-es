= smis crsp
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
La `smis crsp` El comando describe los subperfiles registrados en CIM admitidos por el proveedor SMI-S de NetApp, incluidos los subperfiles de proveedores SMI-S de NetApp.

.Sintaxis
smis crsp

`[-t {*http* | *https*}]`

.Ubicación
`C:\Program Files (x86)\NetApp\smis\pegasus\bin`

.Nivel de privilegio
Un usuario con un nombre de usuario y una contraseña válidos

.Parámetros
* `*[-t {http | https}]*`
+
Protocolo que se utilizará: HTTPS (predeterminado) o HTTP
