= smis slpd
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
La `smis slpd` El comando inicia o detiene el daemon SLP.

.Sintaxis
`smis slpd`

`{*start* | *stop*}`

.Ubicaci√≥n
`C:\Program Files (x86)\NetApp\smis\pegasus\bin`

.Nivel de privilegio
Administrador (Windows)

.Ejemplo
Inicie el daemon SLP:

[listing]
----
smis slpd start
SLPD started.
----
Detenga el daemon SLP:

[listing]
----
smis slpd stop
SLPD (15564) was successfully stopped.
----
