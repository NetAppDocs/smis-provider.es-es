= discos smis
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
La `smis disks` el comando muestra información del disco para los sistemas de almacenamiento. `smis disks` Solo funciona cuando se utiliza con controladoras Data ONTAP 7-Mode.

.Sintaxis
discos smis

`[-t {*http* | *https*}]`

.Ubicación
`C:\Program Files (x86)\NetApp\smis\pegasus\bin`

.Nivel de privilegio
Un usuario con un nombre de usuario y una contraseña válidos

.Parámetros
* `*[-t {http | https}]*`
+
Protocolo que se utilizará: HTTPS (predeterminado) o HTTP



.Ejemplo
La `smis disks` comando y su salida abreviada:

[listing]
----
smis disks
